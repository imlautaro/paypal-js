## Usage

Install package:

```sh
# npm
npm install paypal-js

# yarn
yarn add paypal-js

# pnpm
pnpm install paypal-js
```

Import:

```js
// ESM
import { createClient } from 'paypal-js'

// CommonJS
const { createClient } = require('paypal-js')
```
